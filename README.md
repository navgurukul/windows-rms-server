# RMS - Resource Management System

## What is RMS?

**RMS (Resource Management System)** is a comprehensive platform designed to track, monitor, and manage computing devices distributed to NGOs and donors. The system provides centralized control and real-time visibility over distributed laptop resources, enabling organizations to efficiently oversee their device infrastructure.
